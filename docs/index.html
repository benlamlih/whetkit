The loop

The scale, the knife, the proof.

1 · slim

Weigh it — free

Audit any client config: tokens and dollars per server, cross-server duplicate tools, plugin-shipped servers, and which servers deserve alwaysLoad under Claude Code's tool search — recommended from your real usage traces.

whetkit slim --config ~/.claude.json --recommend-hot --from-traces traces.sqlite3

2 · fix

Cut it — reversibly

Self-correcting curation proposes a metadata overlay, re-evaluates through it, and keeps the best plan by measured results. On the demo server: hit-rate 80% → 100% on all 3 of 3 repetitions, tools 14 → 7, tokens −17%. The origin server is never modified.

whetkit fix --server your-server.json --tasks tasks/ --runs 3

3 · prove

Trust it — or catch it

A/B the raw and reduced views on your own tasks. Real example: a glob-filtered gateway kept the task passing — PASS → PASS — while precision collapsed 100% → 36% with 2 wasted calls per task. No gateway would ever tell you that.

whetkit diff before.json after.json

More measured results: GitHub's 44-tool remote MCP slimmed to 7 tools, tokens/task −73%, tool-selection hit-rate held at 100%. Improvements within run-to-run noise are flagged, not celebrated — ⚠ ranges overlap is printed right on the result.

The whole kit

Eleven commands, one honest pipeline.

slimaudit + shrink your client config's union tool surface ($, dedupe, plugins, alwaysLoad, --share)
doctorten-second lint of one server's surface — no tasks, no API key
inspectwhat the agent actually sees: tools, schemas, token costs
generatedraft eval tasks from the inventory (context-aware, read-only by default)
runlive agentic evals: --runs N mean±range, flaky detection, per-task timeouts, cost
curatebaseline → LLM-proposed overlay → re-eval → before/after report
fixself-correcting curation: propose → eval → revise until it converges

diffbefore/after verdict from two summary files
exportshare a plan as an upstream-PR fix table or gateway JSON
overlayserve any curated view as a real MCP server — reversible, origin untouched
